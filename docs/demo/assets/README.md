# Demo Assets

This directory contains the local media used by the README demo section.

Current files:

- `orbital-inspect-demo-hero.png`
- `orbital-inspect-demo-analyze.png`
- `orbital-inspect-demo-decision.png`
- `orbital-inspect-demo-portfolio.png`
- `orbital-inspect-demo-hero.svg`
- `orbital-inspect-demo-analyze.svg`
- `orbital-inspect-demo-decision.svg`
- `orbital-inspect-demo-portfolio.svg`

PNG files are generated from the live app with:

```bash
cd frontend
npm run demo:assets
```

SVG files are editable storyboard fallbacks.

Replacement rule:

Keep the existing basenames where practical so the README can evolve without structural edits.
